BASIC TAG

PENJELASAN

HTML (HyperText Markup Language) adalah bahasa markup standar yang digunakan untuk membangun struktur dan mengatur konten dalam sebuah halaman web.

Beberapa basic tag yang sering digunakan dalam HTML antara lain:

  • <html>: Tag ini digunakan untuk mengawali dokumen HTML dan berisi semua elemen dalam halaman web.
  • <head>: Tag ini berisi informasi metadata tentang dokumen, seperti judul halaman, pemuatan CSS eksternal, dan skrip JavaScript.
  • <body>: Tag ini berisi konten utama halaman web, seperti teks, gambar, hyperlink, dan lainnya.
  • <h1> hingga <h6>: Tag-heading ini digunakan untuk membuat judul dengan tingkatan yang berbeda.
  • <p>: Tag ini digunakan untuk mengelompokkan paragraf teks.
  • <a>: Tag ini digunakan untuk membuat hyperlink atau tautan ke halaman atau sumber daya lain.
  • <img>: Tag ini digunakan untuk menampilkan gambar di halaman web.
  • <ul> dan <li>: Tag ini digunakan untuk membuat daftar tak-berurutan.
  • <ol> dan <li>: Tag ini digunakan untuk membuat daftar berurutan.
  • <div>: Tag ini digunakan untuk mengelompokkan dan memisahkan bagian-bagian dalam halaman web.
